# 鋒兄AI資訊系統

🔥 智能管理您的影片和圖片收藏，支援智能分類和快速搜尋

## 系統概覽

鋒兄AI資訊系統是一個全功能的多媒體內容管理平台，提供以下核心功能：

### 🖼️ 鋒兄圖片庫
- 智能的精美圖片AI創作管理
- 支援 PNG、JPG、JPEG 格式
- 目前管理 241 張圖片，總大小 625.95 MB
- 智能搜尋和分類功能

### 🎬 鋒兄影片庫  
- 智能的精美人生與世界歷程記錄
- 支援 MP4 格式影片播放
- 包含「鋒兄的傳奇人生」和「鋒兄進化Show」等精彩內容

### 🍎 食品管理系統
- 智能的食品保存及訓練狀態管理
- 自動到期提醒功能
- 支援食品分類和庫存管理
- 目前管理 15 種食品項目

### 📋 訂閱管理系統
- 智能的合理訂閱服務和會員資訊管理
- 自動到期提醒
- 支援多種訂閱服務管理
- 目前管理 24 個訂閱項目

## 安裝說明

1. **環境要求**
   - PHP 7.4 或更高版本
   - Apache/Nginx Web 服務器
   - 支援 .htaccess 重寫

2. **部署步驟**
   ```bash
   # 克隆或上傳文件到 public_html 目錄
   # 確保 Web 服務器指向 public_html 目錄
   # 設置適當的文件權限
   chmod 755 public_html/
   chmod 644 public_html/*.php
   ```

3. **配置**
   - 確保 Apache mod_rewrite 模組已啟用
   - 檢查 .htaccess 文件權限
   - 配置 PHP 錯誤報告（生產環境建議關閉）

## 使用指南

### 導航系統
- **內頁** - 系統儀表板和統計概覽

- **影片庫** - 觀看和管理影片內容
- **食品管理** - 管理食品庫存和到期日
- **訂閱管理** - 管理各種訂閱服務

### 搜尋功能
- 在各個模組中使用搜尋框快速查找內容
- 支援模糊搜尋和關鍵字匹配
- 實時搜尋結果更新

### 響應式設計
- 桌面端：完整側邊欄導航
- 移動端：漢堡菜單導航
- 自適應網格佈局

## API 接口

系統提供 RESTful API 接口：

```
GET  /api?path=stats          - 獲取系統統計
GET  /api?path=gallery        - 獲取圖片列表
GET  /api?path=videos         - 獲取影片列表
GET  /api?path=food           - 獲取食品列表
GET  /api?path=subscription   - 獲取訂閱列表
POST /api?path=search         - 搜尋功能
```
