## 📦 安裝部署

### 環境要求
- PHP 7.4 或更高版本
- MySQL 5.7 或更高版本
- Apache/Nginx Web 服務器
- PDO 和 PDO_MySQL 擴展

### 快速安裝
1. **下載源碼**
   ```bash
   git clone https://github.com/goldshoot0720/laravel20260101.git
   cd laravel20260101
   ```

2. **配置資料庫**
   - 本地測試：`localhost/root/(空密碼)/feng_laravel`
   - 遠端部署：`localhost/feng_laravel/ym0Tagood129/feng_laravel`

3. **執行安裝**
   - 訪問 `/install.php` 進行圖形化安裝
   - 或執行 `php database/migrations.php` 手動安裝

4. **啟動服務**
   - 本地測試：雙擊 `start-local-server.bat`
   - 生產環境：配置 Web 服務器指向 `public_html` 目錄
